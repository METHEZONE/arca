# ARCA — 10분 IR 피칭 스크립트 (투자자용 · 숫자 + 스토리)
**10분 / 13슬라이드 / 한국어 / 솔직·담백 대화체(토스식) + 확실한 IR 숫자**

> 톤 가이드(이승건 토스 "10년의 여정" 키노트 기반): 센척 금지·과장 금지. **고백 + 확신**이 핵심. 솔직하게 인정하고("처음엔 노트앱 만들려고 했어요"), 믿는 건 짧고 단단하게 단언하고("그게 본질이에요"), 비전은 깔끔한 3키워드로("A를 넘어 B로").
> 어미: "~거든요 / ~어요" 대화체 + "~습니다/~이에요" 확신 단언을 섞음. 너무 캐주얼하게 "~더라고요"만 도배하지 말 것.
> 참고 발화: "어차피 어디서든 힘들 거라면, 진정으로 해결하고 싶은 문제에 시간을 쏟아야 합니다. 그게 스타트업의 본질이에요." / "안 해보고 후회하긴 싫다. 하고 망하더라도 납득이 가는 게 낫다."

**시간:** 본편 ~7:30 + 데모 90s + 버퍼 60s = **목표 8:45**, 하드캡 **10:00**.
범례 — 🖥️ 화면 / 🎤 멘트(구어체) / ⏱️ 시간 / ➡️ 전환
