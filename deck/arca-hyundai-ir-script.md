## 🔴 S6 · 라이브 데모 ⏱️ 150s (실연 90s + 무대 버퍼 60s)

**사전 셋업(발표 30분 전):** `apps/macos/dist/ARCA.dmg` 실행 · `~/.arca/settings.json` 키 확인 · `--selftest` green · 마이크 권한 · Slack은 **테스트 채널만**. 운영자 = **발표자 본인(솔로)**.

**라이브 = (a)루프 완수 + (b)단일 ⌥선택만. (c)recap은 다음 슬라이드.**

1. **(0–15s)** "회의 하나만 부탁해 볼게요" → Talk to ARCA → **단일 발화**: *"내일 오후 2시에 데모 준비 회의 잡아줘."*
2. **(15–55s)** 라이브 STT → Claude가 ActionPlan 생성 *(키 없으면 fixture 자동·무중단)* → 결정 카드 등장. *(응답 대기 버퍼 구간)*
3. **(55–75s)** 추천 별표(1번=캘린더) 가리키며: "ARCA 추천은 1번입니다." → **⌥1 한 번** → Calendar.app 실제로 열림.
4. **(75–90s)** "**루프가 닫혔습니다.** 제가 직접 한 건 없는데 일정이 잡혔고, 하단 메모리 로그에 **방금 기록**됐습니다."

**🛟 데모 실패 폴백:** STT/응답이 늦거나 막히면 즉시 → "라이브는 네트워크를 타니, 방금 흐름을 캡처로 보겠습니다" → **폴백 슬라이드(녹화/스샷)**로 전환. fixture 자동 폴백이 있어 카드까지는 거의 항상 뜸. **90초 넘기면 무조건 컷.**

➡️ (덱으로 복귀)

## S7 · 데모 임팩트 + Recap 목업 ⏱️ 55s

- 좌 **방금 일어난 일**: 루프 닫힘 · 절약 시간 · *(방금 생긴 메모리 로그 실제 스샷)*
- 우 **이번 주 recap**(데이터 기반 정적 목업): 인과사슬 · 관계 추천

🎤 "방금 제가 직접 한 일은 없는데 일정이 잡히고 세컨드브레인에 기록됐습니다 — 보시는 이 로그가 **실제로 방금 생긴 것**입니다. 이게 매일 쌓이면 이런 **주간 recap**이 나옵니다. *'이 결정이 이 결과로 이어졌다', '이 사람과 저 사람을 잇자'* — **인간관계의 메타인지**죠. 데이터는 이미 쌓이고 있고, 이 recap 뷰가 **다음 단계**입니다."

➡️ "그래서 우리의 해자는 뭘까요?"

➡️ "그럼 지금까지 뭘 만들었느냐."

## S11 · Ask ⏱️ 30s
🖥️
- 제목: **함께 지켜봐 주세요**
- 이번 단계 = **관계 · 검증**
- 다음 단계 = **디자인 파트너 → 시드**
- (연락 QR / 캘린더 링크)

🎤 "오늘은 금액을 말씀드리는 자리가 아닙니다. 지금 저희에게 필요한 건 **관계와 검증**입니다 — 함께 써보실 **디자인 파트너**, 그리고 그다음이 **시드**입니다. 이 여정을 같이 지켜봐 주세요. 연락 경로 남기겠습니다. 감사합니다."

> Ask 노트: "금액 없음"은 회피가 아니라 **명시적 단계 설계**(관계·검증 → 디자인파트너 → 시드)임을 분명히. → CVC가 "준비 안 됨"이 아니라 "단계가 잡혀 있다"로 읽게.
